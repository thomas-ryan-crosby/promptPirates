⚔️ Tactics

🧪 API-first, then test for hours

Make every action reachable via API so an agent can run thousands of end-to-end scenarios overnight. The escapees are weird sequences, not happy paths — describe adversarial flows explicitly.

🐙 Let Fable break it

Use Fable as the adversary: parallel swarms that genuinely try to stress your system. Overnight suites that took 7h finish in ~1h.

📄 HTML over Markdown

Every deliverable: "make it HTML." Diagrams, visuals, something you'd actually read. Markdown is the agent's default, not yours.

🎣 Free demo, then hook

Build a clickable custom app in 24–48h, hand it over free to click around, then price it. "Where do I sign?" beats any pitch deck.

🧠 Fix it in memory

When the agent repeats a mistake (using a deprecated function, breaking tests), tell it to write the lesson to its memory file. It stops happening.

🔭 Niche, then scale

Hyper-custom software for one narrow vertical → templatize → scale. Everyone serious is converging here.
